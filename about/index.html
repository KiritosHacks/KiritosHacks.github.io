# 此博客搭建的过程

其实很早以前就想搭建一个自己的博客，但因为一些琐事，经常忘记这件事情。

一开始我是想自己用 java 搭一个博客的，但是考虑到代码的工作量以及域名的申请需要一段时间，所以就暂时偷个懒使用现成的框架和主题了。

这个博客就先用来记录我日常遇到的一些问题，以及学习中的一些困难。

后面的话我会自己动手搭建一个博客网站并且记录下来。
